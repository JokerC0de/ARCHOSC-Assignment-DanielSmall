Section 3: Hosting the Website with IIS

The next step would have been to check that IIS was running. IIS is Microsoft’s web server software, and it can be used to host a website from a Windows Server virtual machine. 

Get-Service w3svc

After confirming IIS was running, I would have copied my website files into the C:\inetpub\wwwroot folder so that the website could be served from the VM.

Section 4: Development and Deployment Use

A virtual machine can be used as a separate environment for development, testing or deployment. A development VM is useful for testing changes before they go live, while a deployment VM is used to host the version that users access.

Reflection

Although I was unable to connect to the VM, this lab helped me understand how virtual machines can be used to host and test websites. The process would involve connecting through Remote Desktop, cloning the GitHub repository, checking IIS, and copying the website into the web server folder.

This is useful in industry because cloud platforms often use virtual machines to host applications. Separating development and deployment environments helps reduce the risk of breaking a live system.
