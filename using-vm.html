Accessing and Using a Virtual Machine

What I did

This lab introduced virtual machines. A virtual machine is a separate computer environment that runs on top of physical hardware. It can be used for development, testing and hosting.

Evidence

I tried to access the university vm but it was not working or allowing me to connect.

Reflection

Virtual machines are useful in industry because they allow developers and organisations to run multiple isolated systems without needing separate physical computers. This is used in cloud computing, testing, deployment and server hosting.
